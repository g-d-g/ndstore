Data APIs¶

HDF5 Service¶

POST¶

POST (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/hdf5/(int: resolution)/(int: min_x),(int: max_x)/(int: min_y),(int: max_y)/(int: min_z),(int: max_z)/(int: min_time),(int: max_time)/¶
Synopsis:

Post a 3D/4D region of data for a single channel or multiple channels in HDF5 file format to the server. Form parameters describe the datasets within the HDF5 file.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in NeuroData.
  • channel_name (string) – Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • min_x (int) – Minimum value in the xrange
  • max_x (int) – Maximum value in the xrange
  • min_y (int) – Minimum value in the yrange
  • max_y (int) – Maximum value in the yrange
  • min_z (int) – Minimum value in the zrange
  • max_z (int) – Maximum value in the zrange
  • min_time (int) – Minimum value in the timerange. Optional. Only used for timeseries channels.
  • max_time (int) – Maximum value in the timerange. Optional. Only used for timeseries channels.
Form Parameters:
 
  • CUTOUT – HDF5 group, Post data
  • CHANNELTYPE – HDF5 group, Channel type(image, annotation, probmap, timeseries)
  • DATATYPE – HDF5 group, Data type(uint8, uint16, uint32, rgb32, rgb64, float32)
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Image Slice Service¶

GET XY Slice Cutout¶

GET (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/xy/(int: resolution)/(int: min_x),(int: max_x)/(int: min_y),(int: max_y)/(int: z_slice)/(int: time_slice)/¶
Synopsis:

Download a single image of a specified canonical plane, XY, and specified channel, resolution and bounds. Your browser can load these images.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in NeuroData.
  • channel_name (string) – Channel Name in NeuroData. Optional. If missing will use default channel for the token.
  • resolution (int) – Resolution for the data
  • min_x (int) – Minimum value in the xrange
  • max_x (int) – Maximum value in the xrange
  • min_y (int) – Minimum value in the yrange
  • max_y (int) – Maximum value in the yrange
  • z_slice (int) – Z-slice value
  • time_slice (int) – Minimum value in the timerange. Optional. Only used for timeseries channels.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET /ocp/ca/kasthuri11/image/xy/0/9000,9500/12000,12500/50/ HTTP/1.1
Host: openconnecto.me

Example Response:

HTTP/1.1 200 OK
Content-Type: image/tiff
