Project Info API’s¶

JSON Service¶

GET¶

GET (string: server_name)/ocp/ca/(string: token_name)/info/¶
Synopsis:

Get project information from the server
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

GET /ocp/ca/kasthuri11/info/ HTTP/1.1
Host: openconnecto.me

POST¶

GET (string: server_name)/ocp/ca/json/¶
Synopsis:

Get a HDF5 file from the server
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
Form Parameters:
 
  • JSON – Look at the Tech Sheet
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET /ocp/ca/json/ HTTP/1.1
Host: openconnecto.me
Content-Type: application/json

{
  dataset
  project
  metadata
}

Example Response:

HTTP/1.1 200 OK
Content-Type: application/json

{
  SUCCESS
}

HDF5 Service¶

GET¶

GET (string: server_name)/ocp/ca/(string: token_name)/projinfo/¶
Synopsis:

Post a Numpy file to the server
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET /ocp/ca/kasthuri11/projinfo/ HTTP/1.1
Host: openconnecto.me

Example Response:

HTTP/1.1 200 OK
Content-Type: application/hdf5
