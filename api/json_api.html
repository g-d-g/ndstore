JSON APIs¶

createChannel¶

POST (string: server_name)/ocp/ca/(string: token_name)/createChannel/¶
Synopsis:

Create a list of channels for an existing project using the project token and JSON file.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in NeuroData.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

POST /ocp/ca/test_kat1/ HTTP/1.1
HOST: openconnecto.me
Content-Type: application/json

{
  "channels": {
    "CHAN1": {
      "channel_name": "CHAN1",
      "channel_type": "image",
      "datatype": "uint8",
      "readonly": 0
    },
    "CHAN2": {
      "channel_name": "CHAN2",
      "channel_type": "image",
      "datatype": "uint8",
      "readonly": 0
    }
  }
}

Example Response:

HTTP/1.1 200 OK
Content-Type: application/json

{
  "SUCCESS"
}

deleteChannel¶

POST (string: server_name)/ocp/ca/(string: token_name)/deleteChannel/¶
Synopsis:

Delete a list of channels for an existing project using the project token and a JSON file.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in NeuroData.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

POST /ocp/ca/test_kat1/ HTTP/1.1
HOST: openconnecto.me
Content-Type: application/json

{
  "channels": [
             "CHAN2",
             "CHAN3"
  ]
}

Example Response:

HTTP/1.1 200 OK
Content-Type: application/json

{
  "SUCCESS"
}

autoIngest¶

POST (string: server_name)/ocp/ca/autoIngest/¶
Synopsis:

Create a dataset, project and channels with a JSON file.
Parameters:
  • server_name (string) – Server Name in NeuroData. In the general case this is openconnecto.me.
Form Parameters:
 
  • JSON – Look at the Tech Sheet
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET /ocp/ca/json/ HTTP/1.1
Host: openconnecto.me
Content-Type: application/json

{
  dataset
  project
  metadata
}

Example Response:

HTTP/1.1 200 OK
Content-Type: application/json

{
  SUCCESS
}
