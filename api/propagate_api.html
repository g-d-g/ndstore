Propagate API’s¶

getPropagate¶

GET (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/getPropagate/¶
Synopsis:

Get the propagation state of the channel.
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

setPropagate¶

GET (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/setPropagate/(int: propagate_value)/¶
Synopsis:

Set the propagation state of the channel.
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
  • propagate_value – 0,1,2
  • propagate_value – int
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format
