RAMON API’s¶

RAMON Field Service¶

setField¶

GET (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/setField/(string: ramon_field)/(string/int/float: ramon_value)¶
Synopsis:

Set the value of the RAMON field for the specified channel
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is ocp.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
  • ramon_field (string) – RAMON field. For more details look at the RAMON documents.
  • ramon_value (string/int/float) – Value of the corresponding RAMON field
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Annotation Service¶

GET¶

GET (string: server_name)/ocp/ca/(string: token_name)/(string: channel_name)/(int: annotation_id)/(string: option_args)/(int: resolution)/¶
Synopsis:

Get an annotation to the server
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is ocp.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP. Optional. If missing will use default channel for the token.
  • annotation_id (int) – Id of the annotation to be cut from the database.
  • options_args – Optional Arguments. This can be overwrite, preserve, exception.
  • resolution (int) – Resolution for the annotation
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format
