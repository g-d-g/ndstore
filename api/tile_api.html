Tile API’s¶

You can also view the Tilecache API’s which work similarly for ocptilecache.

getSimpleTile¶

GET (string: server_name)/ocp/catmaid/(string: token_name)/(string: channel_name)/(string: slice_type)/(int: time)/(int: zvalue)/(int: ytile)_(int: xtile)_(int: resolution).png¶
Synopsis:

Get a 512x512 tile from the database
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP.
  • slice_type (string) – Type of Slice cutout. Can be xy/yz/xz
  • time (int) – Time value. Optional. Only possible in timeseries datasets.
  • zvalue (int) – Zslice value.
  • ytile (int) – Y-Tile value. Each tile is 512x512.
  • xtile (int) – X-Tile value. Each tile is 512x512.
  • resolution (int) – Resolution value.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET  /ocp/catmaid/kasthuri11/image/xy/1/1_1_4.png HTTP/1.1
Host: openconnecto.me

Example Response:

HTTP/1.1 200 OK
Content-Type: application/png

getMcfcTile¶

GET (string: server_name)/ocp/catmaid/mcfc/(string: token_name)/(string: channel_name):(string: color_name)/(string: slice_type)/(int: time)/(int: zvalue)/(int: ytile)_(int: xtile)_(int: resolution).png¶
Synopsis:

Get a 512x512 tile from the database
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP.
  • color_name (string) – Color Name. Can be ‘C/M/Y/R/G/B’. Optional If Missing will default to “CMYRGB”.
  • slice_type (string) – Type of Slice cutout. Can be xy/yz/xz
  • time (int) – Time value. Optional. Only possible in timeseries datasets.
  • zvalue (int) – Zslice value.
  • ytile (int) – Y-Tile value. Each tile is 512x512.
  • xtile (int) – X-Tile value. Each tile is 512x512.
  • resolution (int) – Resolution value.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

GET  /ocp/catmaid/mcfc/Thy1eYFPBrain10/Grayscale/xy/500/0_0_3.png HTTP/1.1
Host: openconnecto.me

Example Response:

HTTP/1.1 200 OK
Content-Type: application/png

Synopsis:

Get a 512x512 tile from the database
Parameters:
  • server_name (string) – Server Name in OCP. In the general case this is openconnecto.me.
  • token_name (string) – Token Name in OCP.
  • channel_name (string) – Channel Name in OCP.
  • resolution (int) – Resolution value.
  • xtile (int) – X-Tile value. Each tile is 512x512.
  • ytile (int) – Y-Tile value. Each tile is 512x512.
  • zvalue (int) – Zslice value.
Status Codes:
  • 200 OK – No error
  • 404 Not Found – Error in the syntax or file format

Example Request:

Example Response:

Content-Type: application/png
